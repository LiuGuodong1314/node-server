# node server

#### 介绍
本项目Node前端服务器
+ 支持webpack打包后的前端
+ 其它（未测试）

#### 软件架构
软件架构说明
+ server.js 核心服务器
+ application-dev.yml 开发环境
+ package.json
+ package-lock.json
+ dist

#### 安装教程

1. 将打包后的静态文件放到dist中
2. npm install
3. npm run dev

#### 使用说明

1. 默认首页名称 index.hml 
2. favicon.ico 默认在dist目录下
3. 环境变量   
   + dev 开发环境
   + test 测试环境
   + prod 生产环境
